# Sales-Finance-Analytics-Excel-Project

## 🚀 Project Overview

This project demonstrates an end-to-end Sales and Finance Analytics solution built in Microsoft Excel using Power Query, Power Pivot, Data Modeling, and DAX.

The objective was to transform raw business data into meaningful insights that support decision-making across sales performance, customer analysis, market expansion, and financial reporting.

---

## 📈 Sales Analytics

### Customer Performance Report

📄 [View Report](https://github.com/KhushiSoni-78/Sales-Finance-Analytics-Excel-Project/blob/main/customer%20performance%20report.pdf)

Analyzed customer-wise sales performance across multiple years to identify growth trends, high-value customers, and revenue contribution.

### Market Performance vs Target Report

📄 [View Report](https://github.com/KhushiSoni-78/Sales-Finance-Analytics-Excel-Project/blob/main/market%20performance%20vs%20target.pdf)

Compared actual market sales against predefined targets to evaluate performance gaps and uncover opportunities for improvement.

### Key Insights

- Identified top-performing customers contributing significantly to revenue.
- Evaluated year-over-year sales growth trends.
- Measured market performance against sales targets.
- Highlighted underperforming and high-potential markets.
- Supported data-driven sales and expansion strategies.
